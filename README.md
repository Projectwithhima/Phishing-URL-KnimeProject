# Phishing-URL-KnimeProject

The dataset is used for classification tasks, specifically to distinguish between legitimate and phishing URLs.
The primary problem being addressed is phishing detection. Phishing is a significant cybersecurity threat where attackers deceive individuals into providing sensitive information by masquerading as trustworthy entities.

Data Source:- UCI Repository

Data Overview:

Size - 54 Features and 235,795 instances

Types of Data - Categorical Data, Integer Data and Continuous Data.
